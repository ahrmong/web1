HTML

Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications. Web browsers recrive HTML documents from a web server or from local storage and render them into multimedia web pages. HTML descripbes the strucuture of a web page semantically and originally included cues for the appearance of the document.
